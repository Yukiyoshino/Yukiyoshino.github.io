Web Designer Yuki Yoshino

福岡市在住のWebディレクター、ヨシノユキです。オンラインスクールでWeb制作について学び、仕事の中でのウェブ制作ディレクションに役立てたいと思っています。さらにWeb制作の技術や知識を身につけて、プログラマーとのスムーズな連携により、質の高いウェブサイトの制作に取り組めるようになりたいと考えています。

Skills

Web

目的に沿った効果的なウェブサイト制作に対して、デザインや機能などさまざまな視点からアドバイスをします。

Edit

雑誌や書籍での編集・ライター経験を活かし、コンテンツの内容や見せ方、文章についてもサポートが可能です。

Inbound
